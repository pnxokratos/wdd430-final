How to use the Todo App

  • Use the input to type directly a task and add it
  • The alternative way to add a task is by clicking the "New Task" on the navbar

Options

Each task will have 3 buttons

  • Check: If clicked, the task will update as complete
  • Edit: You can edit your task by click on this button
  • Delete: Delete your task

Filter

There are 3 filters available:

  • All: show all task added
  • Pending: show all task incomplete
  • Complete: show all task completed

Tools used

This is a MEAN Stack Application

  • Database: MongoDb
  • Node Js Framework: Express
  • Javascript Framework: Angular
  • Javascript Runtime: Node Js
  • Icons: FontAwesome
